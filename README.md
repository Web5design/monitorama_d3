monitorama_d3
=============

introductory workshop talk for monitorama about using d3 to monitor bitly clicks

To see this try, in one terminal:

    python server.py
    
Then, in another:

    python reader.py
    
Then, to see the talk, go to http://locahost:8888/talk. To see the examples, look at the endpoints at the end of server.py.
